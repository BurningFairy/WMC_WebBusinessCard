# WMC_WebBusinessCard

A small personal website created for a school project. It introduces me, shows my pets, includes a contact form, and has a simple clicker game.

## Pages / Sections

- **Home**: short introduction and basic information about me
- **My Pets**: pictures and descriptions of my pets
- **Contact**: a contact form to send a message
- **Clicker Game**: a small game where you click to increase your score
